sample question :
0:Write a query  to display all the bank details.

Solution : select * from BANK_INFO;

1.*Write a query which will display the customer id, account type they hold, their account number and bank name.

Solution : select a.customer_id,a.account_type,a.account_no,b.bank_name from ACCOUNT_INFO a join BANK_INFO b on (a.ifsc_code = b.ifsc_code);

2.*Write a query which will display the customer id, account type and the account number of HDFC customers who registered after 12-JAN-2012 and before 04-APR-2012.

Solution : select customer_id,account_type,account_no,bank_name from ACCOUNT_INFO natural join BANK_INFO where bank_name = 'HDFC' and registration_date between '01/13/2012' and '04/03/2012';

3.*Write a query which will display the customer id, customer name, account no, account type and bank name where the customers hold the account.

Solution :select customer_id,account_no,account_type,bank_name,branch_name from ACCOUNT_INFO natural join BANK_INFO join CUSTOMER_PERSONAL_INFO using(customer_id);

4.*Write a query which will display the customer id, customer name, gender, marital status along with the unique reference string and sort the records based on customer id in descending order. 
Hint:  Generate unique reference string as mentioned below
:
CustomerName_Gender_MaritalStatus
 Example 
 C-005           KUMAR              M                 SINGLE            KUMAR_M_SINGLE

Use ""UNIQUE_REF_STRING"" as alias name for displaying the unique reference string."

Solution: select customer_id,customer_name,gender,marital_status,customer_name||' '||gender||' '||marital_status"unique reference string" from CUSTOMER_PERSONAL_INFO order by customer_id desc;

5.*Write a query which will display the account number, customer id, registration date, initial deposit amount of the customer
 whose initial deposit amount is within the range of Rs.15000 to Rs.25000.

Solution : select account_no,customer_id,registration_date,initial_deposit from ACCOUNT_INFO where initial_deposit between 15000 and 25000;


6.*Write a query which will display customer id, customer name, date of birth, guardian name of the customers whose name starts with 'J'.

Solution : select customer_id,customer_name,date_of_birth,guardian_name from CUSTOMER_PERSONAL_INFO where customer_name like 'J%';

7.Write a query to display the customer id, customer name, date of birth, guardian name, contact number, 
mail id, reference name who has been referenced by 'RAGHUL'.

Solution: select customer_id,customer_name,date_of_birth,guardian_name,contact_no,mail_id,reference_acc_no from CUSTOMER_PERSONAL_INFO natural join CUSTOMER_REFERENCE_INFO where reference_acc_name = 'RAGHUL';

8.*Write a query which will display the customer id, customer name, date of birth, Marital Status, Gender, Guardian name, 
contact no and email id of the customers whose gender is male 'M' and marital status is MARRIED.

Solution: select customer_id,customer_name,date_of_birth,marital_status,gender,guardian_name,contact_no,mail_id from CUSTOMER_PERSONAL_INFO where gender='M' and marital_status='MARRIED';

9.*Write a query which will display the customer id, customer name, guardian name, reference account holders name of the customers 
who are referenced / referred by their 'FRIEND'.

Solution: select customer_id,customer_name,guardian_name,reference_acc_name from CUSTOMER_PERSONAL_INFO natural join CUSTOMER_REFERENCE_INFO where relation='FRIEND';



10.*Write a query to display the customer id, account number and interest amount in the below format with INTEREST_AMT as alias name
 Sort the result based on the INTEREST_AMT in ascending order.  <BR/>Example: 
$5<BR/>Hint: Need to prefix $ to interest amount and round the result without decimals.
 
Solution: select customer_id,account_no,to_char(round(interest,0),'$9') as interest_amt from ACCOUNT_INFO order by interest;

11.*Write a query which will display the customer id, customer name, account no, account type, activation date,
 bank name whose account will be activated on '10-APR-2012'

Solution : select customer_id,customer_name,account_no,account_type,activation_date,bank_name from CUSTOMER_PERSONAL_INFO natural join ACCOUNT_INFO join BANK_INFO using(ifsc_code) where activation_date = '04/10/2012'; 

12.*Write a query which will display account number, customer id, customer name, bank name, branch name, ifsc code
, citizenship, interest and initial deposit amount of all the customers.

Solution : select account_no,customer_id,customer_name,bank_name,branch_name,ifsc_code,citizenship,interest,initial_deposit from CUSTOMER_PERSONAL_INFO natural join ACCOUNT_INFO join BANK_INFO using(ifsc_code);

13.*Write a query which will display customer id, customer name, date of birth, guardian name, contact number,
 mail id and reference account holder's name of the customers who has submitted the passport as an identification document.

Solution: select customer_id,customer_name,date_of_birth,guardian_name,contact_no,mail_id,reference_acc_name from CUSTOMER_PERSONAL_INFO natural join CUSTOMER_REFERENCE_INFO
 where identification_doc_type = 'PASSPORT';

14.*Write a query to display the customer id, customer name, account number, account type, initial deposit, 
interest who have deposited maximum amount in the bank.

Solution: select customer_id,customer_name,account_no,account_type, initial_deposit ,interest from ACCOUNT_INFO natural join CUSTOMER_PERSONAL_INFO where initial_deposit = ( select MAX(initial_deposit) from ACCOUNT_INFO);

15.*Write a query to display the customer id, customer name, account number, account type, interest, bank name 
and initial deposit amount of the customers who are getting maximum interest rate.


Solution : select customer_id,customer_name,account_no,account_type,interest,bank_name,initial_deposit from CUSTOMER_PERSONAL_INFO natural join  ACCOUNT_INFO join BANK_INFO using(ifsc_code) 
where interest = ( select MAX (interest) from ACCOUNT_INFO);
16.Write a query to display the customer id, customer name, account no, bank name, contact no 
and mail id of the customers who are from BANGALORE.

Solution : select customer_id,customer_name,account_no,bank_name,contact_no,mail_id from ACCOUNT_INFO natural join BANK_INFO join CUSTOMER_PERSONAL_INFO using(customer_id) where address like '%BANGALORE';


17.Write a query which will display customer id, bank name, branch name, ifsc code, registration date, 
activation date of the customers whose activation date is in the month of march (March 1'st to March 31'st).

select customer_id,bank_name,branch_name,ifsc_code,registration_date,activation_date from BANK_INFO join ACCOUNT_INFO using(ifsc_code) where activation_date between to_date ('01-MARCH-2012','DD-MONTH-YYYY') and to_date ('31-MARCH-2012','DD-MONTH-YYYY');
